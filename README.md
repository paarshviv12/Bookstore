# Bookstore Interface 📚

A modern, front-end web interface for an online bookstore. It dynamically presents a variety of popular books across different literary genres in an organized format.

## 🌟 Features
- **Book Display:** Visually distinct sections for different book genres.
- **Responsive Layout:** Smooth layout adaptation across various device screen sizes.
- **Interactive Elements:** Clean UI hover states for enhanced browsing experience.

## 🛠️ Built With
- **HTML5:** Semantic architecture.
- **CSS3:** Flexbox and Grid based structures for book displays.

## 🚀 Getting Started
To view this project locally, simply clone the repository and open the HTML file.

### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/paarshviv12/Bookstore.git
   ```
2. Open `bookstore.html` in your web browser.
